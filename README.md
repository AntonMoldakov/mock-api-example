# About

## Getting started

1. Install dependencies with `yarn install`
2. Start the development server with `yarn dev`
3. Generate api `yarn api`

## To Run Mock Api

1. Change env VITE_USE_MOCK_API to true
2. Generate handlers `yarn mock:generate`
